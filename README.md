# 網紅邀約文案產生器

這是一個純前端靜態網頁工具，可直接打開 `index.html` 使用。

功能：

- 輸入網紅名稱，自動依文字偵測語言。
- 用 `{name}` 套入預先設定好的文案。
- 可新增、編輯、暫停、刪除多個文案。
- 每個文案和全部文案都能一鍵複製。
- 設定存在目前瀏覽器，也可用 JSON 匯入/匯出到其他裝置。

## 使用方式

直接用瀏覽器打開 `index.html`，或部署到 GitHub Pages / Netlify / Vercel 這類靜態網站服務。

## 隱私提醒

網頁本身可以放到 GitHub；你在網頁裡新增的私人文案會存在該裝置瀏覽器的 localStorage，不會自動寫進 GitHub。

如果把真實文案直接改進 `index.html` 的預設模板，部署後能看到網頁原始碼的人也可能看到那些文案。私人使用時，建議先部署空工具，再用網頁裡的設定區新增文案。

## 跨裝置

手機和電腦的瀏覽器設定是分開的。要同步文案時，在舊裝置按「匯出設定」，到新裝置貼上後按「匯入設定」。
